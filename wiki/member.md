####  部门管理

---
##### 部门列表

| key | value |
| --- | ----- |
| url | http://domain/department/list |
| method | get |
| cookie | uid,token |
| get params | index=0&limit=10|

######返回结果：
```
{
    "code": 0,
    "msg": "",
    "total":23,
    "data": [
        {
            "id": 1,
            "name": "总办"
        },
        {
            "id": 2,
            "name": "事场部"
        },
        ...
    ]
}
```

##### 添加、修改部门

| key | value |
| --- | ----- |
| url | http://domain/department/set |
| method | post |
| cookie | uid,token |
| post params | id:xxx(索引,没有此字段或空值表示新增)<br/>name:xxxx(部门名称)|

######返回结果：
```
{
    "code": 0,
    "msg": ""
}
```

##### 删除部门

| key | value |
| --- | ----- |
| url | http://domain/department/delete |
| method | post |
| cookie | uid,token |
| post params | id:xxx|

######返回结果：
```
{
    "code": 0,
    "msg": ""
}
```
